{{- /* Single source of truth for "should this page be noindex'd?". Consumed by head.html (robots meta) and the sitemap templates (layouts/sitemap.xml + layouts/sitemapindex.xml, which skip the URL). Rules: - 404 pages - any page with `robotsNoIndex: true` front matter - THIN tag term pages: a tag grouping fewer than 3 posts. ~70% of tags on this site have a single post; that long tail wastes crawl budget and reads as thin/duplicate content without ranking for anything. noindex,follow lets Google keep crawling the links while dropping the page from the index. Stays INDEXABLE: strong tag hubs (>=3 posts), the /tags/ list page itself (Kind "taxonomy"), and every `columns` page. Threshold lives here so it can be tuned in one place. */ -}} {{- $thinTagMin := 3 -}} {{- $thinTag := and (eq .Kind "term") (eq .Data.Plural "tags") (lt (len .Pages) $thinTagMin) -}} {{- return (or (eq .Kind "404") .Params.robotsNoIndex $thinTag) -}}
