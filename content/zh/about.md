## 📍 我从哪里来

2001 年，我出生于中国（今年 <span id="age-counter">—</span> 岁）。和大多数人一样，我沿着既定的轨道走了很久 —— 读书、考试、进入大学、学习技术。

<script>
(function() {
  function calcAge() {
    var birth = new Date(2001, 2, 15); // 2001-03-15 (month is 0-indexed)
    var now = new Date();
    var age = now.getFullYear() - birth.getFullYear();
    var m = now.getMonth() - birth.getMonth();
    if (m < 0 || (m === 0 && now.getDate() < birth.getDate())) age--;
    return age;
  }
  var el = document.getElementById('age-counter');
  if (el) el.textContent = calcAge();
})();
</script>

但真正的转折点发生在 **2024 年**。那一年，我做了一个决定：**走出舒适区，用脚步和好奇心重新认识这个世界**。

从香港到吉隆坡，从新加坡到曼谷，从博卡拉特到加德满都，从东京到首尔……12 个月，9 个国家，35+ 城市。这不是"间隔年"，而是一场**主动的人生实验**。

<blockquote>
"如果说 2025 年初的我是一个地理上的探索者，那么现在的我，更像是一个在思想旷野中深潜的宇航员。"
</blockquote>

## 🧭 我现在在做什么

### AI 创业与开源

我是一名 AI 创业者，也是开源社区的活跃贡献者。我相信 **AI + Human = Superhuman**，并致力于构建人机协作的新范式。

我参与贡献的项目包括：
- **OpenIM** - 开源即时通讯解决方案
- **OpenKF** - 知识框架项目
- **Sealos** - 云操作系统

### 写作与思考

我通过这个博客记录我的思考、项目拆解、技术探索和人生感悟。写作对我来说，是一种**与自己对话**的方式。

我的写作围绕四条主线展开：
- **AI 与技术** - 技术趋势、AI 应用、开源项目
- **成长与认知** - 学习方法、思维模型、自我迭代
- **旅行与观察** - 跨文化体验、城市观察、社会叙事
- **项目与实践** - 创业历程、产品思考、实战经验

### 徒步与行走

我是一个狂热的徒步爱好者。截至目前，我已经完成了 **400+ 公里** 的徒步旅程。

行走对我来说，是一种**移动的冥想**。在山野之间，在海拔 5416 米的 Tilicho 湖畔，我找到了与内心对话的空间。

## 💡 我为何值得关注

### 1. 第一手经验的执着

我坚信**Raw Data Priority**（一手数据优先）。我不满足于二手知识，而是亲自去体验、去验证、去记录。

这让你从我这里读到的内容，都带着**真实的温度**。

### 2. 跨文化的观察视角

在多个国家和地区的长期生活经历，让我拥有了**跳出单一叙事**的能力。我既能理解中国的技术生态，也能观察东南亚的生活哲学，还能与全球的 AI 社区同频共振。

### 3. 理性与感性的融合

我的操作系统是：**在极致理性中寻找感性的栖息地，在混乱的体验浪潮中校准理性的航向**。

这意味着你从我这里读到的内容，既有逻辑的严谨，也有人性的温度。

### 4. 持续进化的心态

我不害怕承认自己的局限，也不羞于展示自己的困惑。我相信**成长来自于直面自己的无知**。

## 🌉 我的核心议题

如果要用一句话概括我当前的人生议题，那就是：

> **理解世界，构建自我，在技术与人性之间搭建桥梁。**

我在探索：
- 如何让 AI 成为人类认知的延伸，而非替代
- 如何在全球化退潮的时代，做一个连接者
- 如何在追求效率的技术世界里，保留人性的柔软
- 如何在"知道"与"做到"之间，搭建行动的桥梁
