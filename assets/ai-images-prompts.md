## Usage Notes / 使用说明

1. These prompts can be used with AI image generation tools like:
   - Midjourney
   - DALL-E 3
   - Stable Diffusion
   - Adobe Firefly

2. For consistent style, use the same seed or style reference across all images.

3. Recommended aspect ratio: 16:9 or 3:2 for blog post images.

4. Consider adding subtle Chinese/English text overlays for key concepts if desired.

5. Ensure images are optimized for web (compressed, appropriate resolution).

representations can be created using geometric shapes and gradients in the
