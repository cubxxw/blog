## Codebase Patterns
- Hugo 自定义 partial 放在 layouts/partials/ 会自动覆盖主题 themes/PaperMod/layouts/partials/ 中同名文件
- About 类型 `type: page` 的页面直接在根目录，breadcrumbs 模板通过 .Parent.Permalink 遍历路径段时无中间段
- CSS 覆盖文件在 assets/css/extended/ 目录，文件名任意，会自动被主入口引入
- hugo build（无参数）等同于完整构建，用于验证模板渲染结果
- 深色模式使用 .dark 类（PaperMod 兼容）和 [data-theme="dark"] 双选择器
- `overflow-x: clip` 而非 `hidden` 不会创建新的滚动容器，不会导致 position:sticky 失效
- 文章正文宽度通过 `.post-single .post-content > [元素类型]` 的直接子选择器控制，可对段落与媒体设置不同 max-width 实现 bleed 效果
- Playwright 需要 @types/node + tsconfig.json 才能通过 TypeScript 类型检查；`npx playwright install --with-deps chromium` 在 CI 中安装浏览器
- package.json 中 Playwright 版本锁定为 ^1.52.0，实际安装 1.59.1
- PaperMod 主题用 body.dark 类切换深色模式，不使用 html[data-theme]
- sticky header 按钮 Playwright 报 outside-of-viewport：用 page.evaluate(() => el.click()) 绕过
