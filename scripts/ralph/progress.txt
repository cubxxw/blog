## Codebase Patterns
- layouts/partials/article/ 目录用于存放文章详情页专属 partial（marginalia、editorial-tools 等），Hugo 自动查找
- i18n key 中可用点号分隔命名空间（如 marginalia.principal），YAML 中 id 字段写完整 key 即可
- Hugo .ReadingTime / .WordCount 在 partial 中直接可用，无需额外计算
- :lang(zh) / :lang(en) CSS 选择器可用于 partial 中的样式隔离，与 Hugo {{ if eq .Lang "zh" }} 模板条件配合使用
- CSS 覆盖文件在 assets/css/extended/ 目录，文件名任意，会自动被主入口引入
- hugo build（无参数）等同于完整构建，用于验证模板渲染结果
- PaperMod 主题用 body.dark 类切换深色模式（不使用 html[data-theme]），但自定义 CSS token 文件中须用 html.dark 匹配
- 语言切换通过 :lang(zh) / :lang(en) CSS 伪类实现，Hugo 生成的 html 标签含 lang 属性
- tsconfig.json 需要 "module": "CommonJS" + "moduleResolution": "node" 才能正确解析 @playwright/test
- package-lock.json 被 .gitignore 忽略（存在 netlify/functions/package-lock.json），不要尝试 git add package-lock.json
- layouts/_default/single.html 覆盖 PaperMod 的同名模板，直接放在 layouts/_default/ 即可生效，无需改主题源码
- EN 文章默认 URL 无语言前缀（/ai-technology/posts/xxx/），ZH 文章 URL 含 /zh/ 前缀
- CSS Grid 三栏布局：.article-layout 设 display:grid + grid-template-columns:repeat(12,minmax(0,1fr))，子元素用 grid-column 定位
